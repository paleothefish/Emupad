# Emupad

### The website to our project can be found [here](https://paleothefish.github.io/Emupad/index.html).
It's a launchpad! With an emulator! Introducing the Emupad! Play your favorite Game Boy and Game Boy Color* games with the decade old CCS3200 launchpad. 

<sub>*Disclaimer: This is a university project intended to understand embedded systems and is in no shape or form intended to be a commercial product.)<sub>
